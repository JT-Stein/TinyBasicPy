# TinyBasicPy

A Tiny BASIC interpreter written in Python.
The syntax is inspired by [TinyBasic v2.4](https://tinybasic.de/language.htm) but has minor alterations.

## Syntax
**Variable Types**
* CHAR
* BYTE
* WORD
* INTEGER
* LONG
* FLOAT

**External File Management**
* LOAD
* STORE

**Loops**
* DO.. LOOP
* FOR.. NEXT

**I/O**
* PRINT
* INPUT

**Mathematical Functions**
* DEG()
* RAD()
* COS()
* TAN()
* ATN()
* SQR()
* EXP()
* LOG()
* POW()
* ABS()

**Date/Time**
* DATE
* DATE.DAY
* DATE.MONTH
* DATE.YEAR
* TIME
* TIME.SECOND
* TIME.MINUTE
* TIME.HOUR

**Console**
* CLS

**Data Visualization**
* PLOT.. NEXT
